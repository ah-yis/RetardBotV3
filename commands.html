Welcome to the commands list!

Before you read:

Remember, the prefix for each command is "-".
Also, make sure you see this before you start going through the commands:

[] = Required arguments
{} = Optional arguments
int = Any integer value.
target = Any user @-pinged in the command.
string = Any string value.

Additional context will probably be provided in the command's information.
